There is a link to access the amazon page.

**URL**: https://ojala.livre.xyz

## 🚀 Features

-   Blogs are written with Perplexity

## ⚙️ Installation

Step-by-step instructions to set up locally:

```bash
git clone https://github.com/Gaet9/ojala.git
cd ojala
npm install
npm run dev
```
